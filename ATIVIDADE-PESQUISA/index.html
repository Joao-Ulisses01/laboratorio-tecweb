ELEMENTOS DO HTML:

Os elementos do HTML são as partes que formam a estrutura de uma página web. Eles são representados por “tags”, que indicam ao navegador como exibir cada tipo de conteúdo, como títulos, parágrafos, imagens e links. Cada elemento tem uma função específica, ajudando a organizar e dar significado ao conteúdo. Assim, o HTML permite que o navegador entenda e mostre a página de forma correta.

ELEMENTOS BÁSICOS

  • DOCTYPE! html- Define a versão do html
  • html- Começa o documento
  • head- Configurações, não aparece na tela
  • body- Tudo que apareçe no site
  • h1- Título
  • p- Parágrafo
